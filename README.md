## Setup API

Access the folder:

```sh
$ cd go-clean-arch-template
```

Create `.env.development` file:

```sh
$ cp .env.example .env.development
```

> Update the variable values as needed.

Configure `/etc/hosts`:

```sh
# Mac and Linux
# /etc/hosts

# Windows
# C:\Windows\System32\drivers\etc\hosts

127.0.0.1 host.docker.internal
```

## Run API

Up Container:

```sh
$ docker-compose up -d
```

Access container:

```sh
$ docker exec -it barber-api /bin/bash
```

Install dependencies:

```sh
$ go mod tidy
```

Run API:

```sh
$ make dev
```

## Scripts

Build API:

```sh
$ make build
```

Start API after build:

```sh
$ make start
```

Upgrade dependencies:

```sh
$ make upgrade-dependencies
```

Generate coverage:

```sh
$ make test-coverage
```

Generate docs:

> You need to install [swag](https://github.com/swaggo/swag) to use the CLI.

```sh
$ make generate-docs
```
